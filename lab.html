Lab Info

I love all things tech, I decided to build a home lab to use for learning new skills and challenging myself. My lab has grown since it inception in 2020. I have opened my lab up to friends that want to enter into the IT field. I provide them with personal mentorship to learn IT skills such as Linux, Ansible, Git, Python, Security and anything else that they want to learn. The details about the backend services and physical hardware can be found below.

Physical Hardware

  1. Mac Pro 4- TrueNAS
    • CPU - Dual Intel Xeon E5520 2.27GHz 4 core
    • Ram - 24 GB DDR3
    • 128 GB SSD - for OS
    • 3 x 3 TB HDD - configured as raid for storage
  2. Dell Precision T7810 - ESXi
    • CPU - Dual Intel Xeon E5-2630 16 core
    • Ram - 128 GB DDR4
    • 128 GB SSD - for OS ESXi
    • 1 TB HDD - Backup storage for VM's
  3. Orbi
    • Orbi Mesh network - RBK53
    • 1 x Router
    • 2 x Satelites
    • 3 Gbps
  4. Rasberry Pi's
    • 2 x Raspbery Pi 4 model b
    • 1 x 2 GB model
    • 1 x 4 GB model

BackEnd Services

  1. TrueNAS
    • Provides Datastore for ESXi Host.
    • Configured as iSCSI for connection.
  2. ESXi
    • Provides hypervisor to create and use virtual machines.
  3. xWiki
    • Provides Learning projects and other documentation.
    • Service also provides task manager so students can work on requested tasks at any time.
  4. Ansible
    • Used to configure basic security configuration, user accounts and other settings on new servers.
  5. DNS
    • Provides DNS service using domain starwars.lab.
    • Router configured to point to internal DNS and then query public DNS servers. IE DNS server does not need to be configured on each server.
  6. Gitlab
    • Provides code repositories for learning projects and other projects.
    • Access is restricted by groups to various projects in Gitlab.
  7. Linux Practice
    • Pre-configured VM to allow students to complete the challenges on the RHCSA practice and Linux Challenges projects.
    • OS is Rockey 9.
  8. NextCloud
    • Provides storage for critical files. Allows easy access to upload/download files dirrectly from a web browser or mobile application.
  9. StarKiller
    • Provides VPN detection using my custom built python solution.
  10. Disaster Recovery
    • In the event that ESXi fails. Build new ESXi host, mount the datastore, reregister virtual machines. Downtime: less than 30 mins.
    • In the event that a disk fails on TrueNAS, replace the disk. Downtime: None.
    • In the event that TrueNAS OS fails. Build new TrueNAS host, remount the disks. Downtime: 15 mins.
